Hugo + Obsidian 目录设计说明

目标

这套目录设计的目标是把 Obsidian 的笔记管理能力和 Hugo 的静态站点生成能力解耦开来：

  • Obsidian 负责管理整个笔记仓库
  • Hugo 只负责把博客相关的 Markdown 转成静态网页
  • GitHub Pages 只负责发布 Hugo 生成的网站

这样做的好处是，博客内容和其他私人笔记、项目笔记、日记不会混在一起，同时 Hugo 站点的职责也保持单一。

设计原则

目录设计遵循下面几条原则：

  • notes/ 是 Obsidian Vault 根目录，也是日常使用的主目录
  • blog/ 是公开博客内容源，只放准备发布的内容
  • site/ 是 Hugo 工程目录，只负责构建和发布
  • 非博客内容默认不进入 Hugo 发布链路
  • 第一版用相对路径软链接连接 blog/ 和 site/

推荐目录结构

notes/
  .obsidian/
  inbox/
  daily/
  knowledge/
  project/
  blog/
    content/
      posts/
      pages/
    assets/
      images/
  site/
    content/
      posts -> ../../blog/content/posts
      pages -> ../../blog/content/pages
    static/
      images -> ../../blog/assets/images
    .github/
      workflows/
        hugo.yml
    hugo.toml

各目录职责

notes/

这是 Obsidian 的 Vault 根目录。你在 Obsidian 里应该直接打开这个目录，而不是只打开 blog/ 或 site/。

初级阶段学习计划

1. 阶段定位

适用时长：第 1-12 周

适用对象：

  • 刚开始接触冥想
  • 很难稳定坐下
  • 经常走神、烦躁、困倦
  • 不清楚“怎么练才算对”

这个阶段的核心，不是追求平静、空白、轻盈感，而是建立最基础的训练回路：

注意偏离 -> 觉察到偏离 -> 轻轻回来

2. 阶段目标

  • 建立稳定的练习习惯
  • 找到一个固定的注意锚点
  • 学会处理走神、困倦、烦躁
  • 初步建立身体觉察
  • 不再把“走神”当成失败

3. 训练主线

本阶段主线：

呼吸专注 -> 简短身体扫描 -> 少量开放觉察预备

比例建议：

  • 呼吸专注：70%
  • 身体扫描：25%
  • 开放觉察预备：5%

4. 12 周周计划

第 1 周

重点：先建立出现频率，不要求状态好。

每日内容：

  • 呼吸专注 5 分钟
  • 练习结束后静坐 1 分钟

要求：

  • 每周 5-7 天
  • 固定一个时间段
  • 固定一个练习位置

第 2 周

重点：让“坐下开始练”变得更自然。

每日内容：

  • 呼吸专注 5-8 分钟
  • 练习结束后静坐 1 分钟

要求：

  • 每周 5-7 天
  • 继续固定时间与地点
  • 呼吸位置只选一个，例如鼻尖、胸口或腹部

第 3 周

重点：从能坐下，进入能发现走神。

高级阶段学习计划

1. 阶段定位

适用时长：1 年以上

高级阶段不等于神秘体验更多，也不等于练得更久。它更看重三件事：

  • 稳定性
  • 可迁移性
  • 一致性

真正的重点是：觉察是否已经进入生活。

2. 阶段目标

  • 长期维持正式练习
  • 深化开放觉察与慈悲训练
  • 在压力、冲突、疲劳、决策中保持觉察
  • 让生活本身成为主要训练场
  • 逐渐形成更稳定的平衡心

3. 训练主线

本阶段主线：

专注稳定维护 -> 开放觉察深化 -> 慈悲与平等心 -> 高密度日常整合

比例建议：

  • 呼吸专注：25%
  • 开放觉察：30%
  • 慈心 / 慈悲 / 平等心：20%
  • 日常整合：25%

4. 长期维护框架

模块一：正式练习维护

每周建议：

  • 呼吸专注 2 次，每次 20-30 分钟
  • 开放觉察 2-3 次，每次 20-30 分钟
  • 慈心 / 慈悲 / 平等心 1-2 次，每次 15-25 分钟

维护原则：

  • 不追求不断加时长
  • 保持结构稳定比偶尔长练更重要

模块二：日常高频整合

每天建议：

  • 工作开始前 1 次停顿
  • 工作切换时 2-3 次呼吸觉察
  • 吃饭时 1 次身体与味觉觉察
  • 走路时 1 次脚步觉察
  • 冲突或压力来时主动暂停若干次

每次只需 10 秒到 3 分钟

冥想学习指南：从入门到进阶与成熟

一、引言

1.1 为什么要学习冥想

冥想并不是某种神秘修行，而是一类可以被系统学习和反复训练的心智练习。美国国家补充与整合健康中心（NCCIH）在官方综述中指出，冥想与正念相关练习对压力、焦虑和抑郁症状可能有帮助，但效果会受到练习方式、课程质量、个体差异和使用场景影响。[1]

从神经科学视角看，部分研究发现，长期冥想练习与大脑功能和连接模式的变化有关，例如默认模式网络（DMN）活动差异、情绪加工相关脑区反应变化，以及注意监测和自我调节相关机制的改变。[2][3][4][5] 这些发现并不意味着冥想可以被神化成万能方法，但足以说明：它不是空泛口号，而是一种有实证基础的训练路径。

更重要的是，冥想并不只发生在打坐时。它真正的价值，通常体现在生活中是否更容易察觉自动反应，是否能在压力、冲突、疲劳和情绪波动中保留一点观察和选择的空间。[5][6]

1.2 冥想常见误解

在开始之前，有必要先澄清几个常见误解。

误解一：“冥想就是把脑子练空。”
不是。冥想的核心不是压制念头，而是提升对念头、身体感觉和情绪活动的觉察能力。注意力跑掉后，能够看见它并回来，这个过程本身就是训练内容。[2]

误解二：“冥想必须盘腿坐很久。”
不是。标准化正念课程 MBSR 包含坐姿练习、身体扫描、行走练习和温和伸展，不要求固定采用某一种姿势。姿势服务于练习，不是练习本身。[7]

误解三：“冥想是万能疗法。”
不是。NCCIH 明确提醒，冥想不能替代必要的医疗或心理治疗；同时，也不是所有人都只会获得正面体验，部分练习者可能出现焦虑加重、痛苦记忆浮现或其他不适。[1][8][9]

误解四：“我老是走神，说明我不适合冥想。”
不是。走神不是失败。研究与实修框架都强调，发现走神并把注意力带回来，本身就是入门阶段的核心训练。[2][10]

1.3 本文的结构与阅读方式

本文先说明冥想的定义、类型和核心能力，再讨论其作用、机制与边界，然后给出三套常见阶段框架，最后进入实操方法、常见困难与资料建议。

如果你是初学者，建议按顺序阅读。先建立概念，再进入方法，会更容易避免把冥想误解成“追求特殊体验”的活动。

二、概念基础

2.1 冥想的定义

现代心理学与认知科学视角

在现代研究语境中，冥想通常被理解为一组通过训练注意力、觉察和心理调节过程来影响体验与行为的练习。Lutz、Slagter、Dunne 和 Davidson 在经典综述中指出，冥想的核心机制之一是对注意的调节与监测，即学习如何维持注意、发现偏离，并在不自动卷入的情况下重新定向。[2]

美国心理学会（APA）的科普页面则把正念冥想概括为一种帮助人们以更当下、更少评判的方式接触经验的练习路径。[11]

东方传统修习语境

在东方传统中，冥想通常被理解为对“心如何运作”的训练。其重点不只是放松，而是通过长期练习培养稳定的注意力、清晰的觉察力以及较少被情绪和执着牵引的能力。传统中的“止”偏向专注与安定，“观”偏向洞察与如实觉察。

无著菩萨在《大乘阿毗达磨集论》中提出的“九住心”，是东方传统中较具代表性的注意力渐进训练框架之一。现代一些实修体系，尤其是十阶段模型，会把它作为古典依据之一。[10]

冥想与正念（Mindfulness）的关系与区别

“冥想”是更大的上位概念，包含多种练习方式；“正念”既可以指一种心理品质，也可以指一类具体练习。卡巴金将正念定义为：有意识地、在当下、并以不评判的方式关注经验。[12]

因此，更准确的说法是：正念是冥想中的重要分支，但并不等于全部冥想。专注训练、开放觉察训练、慈心与慈悲训练都可能与正念有关，但它们的直接训练目标并不完全相同。[2][6]

2.2 冥想的主要类型

依据当前研究中最常见的分类，冥想通常可分为三大主要类型：[2][6]

专注注意（Focused Attention, FA）

把注意力放在单一对象上，例如呼吸、腹部起伏、鼻尖触感或脚步。当注意力偏离时，察觉，再回来。它主要训练注意维持、冲突监测和分心后的回收能力，是最典型的入门路径。[2]

开放监测（Open Monitoring, OM）

不再把注意力死守在一个点上，而是对呼吸、念头、声音、情绪和身体感觉保持开放、非反应式的觉察。它要求练习者既清醒又不过度抓取，因此通常建立在一定专注力基础上。[6]

慈心 / 慈悲冥想（Loving-kindness / Compassion）

通过有意识地培养对自己和他人的善意、温和、关怀与不敌对反应，来改变情绪和关系模式。它在现代研究里常与 FA、OM 并列为三大主要冥想类型之一。[6]

正念减压（MBSR）体系

MBSR 由 Jon Kabat-Zinn 于 1979 年在马萨诸塞大学医学院创立，是当前研究和应用最广的标准化冥想课程之一。典型结构为 8 周，课程中通常包含坐姿冥想、身体扫描、正念行走和温和伸展。它不是单一技巧，而是把多种练习整合进一个可复制的训练框架。[7][12]

  • 适用人群：35 岁左右，希望通过每天 1 小时高效锻炼，达到强健腰椎、保护脆弱膝关节、舒展筋骨并有效抗初老的人群。
  • 当前身体状态：腰椎需强化维稳；膝关节极度脆弱（伴随弹响/酸痛，绝对禁忌深蹲、跳跃）。
